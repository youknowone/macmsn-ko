Create your own emoticons

Show All

You can create emoticons for a personal account, but not for a corporate account.

Important

You can use the following file formats to create custom emoticons: JPEG, BMP, PNG, GIF, and Animated GIF. To be used as an emoticon, a file must be 150 KB or smaller. Animated GIFs must also be 50 x 50 pixels or smaller.

Do any of the following:

Create a custom emoticon

  1. Sign in to your personal accountAn account on the Windows Live network, including all e-mail addresses ending in "@live.com", "@msn.com", and "@hotmail.com". With a personal account you can communicate with other contacts on the Windows Live and Yahoo networks..

  2. On the Edit menu, click Emoticons.

  3. At the bottom of the Emoticons screen, click .

  4. Click Select File, and then locate the picture or animation on your computer.

  5. Under Type a shortcut, enter a keyboard shortcut that you want to use when adding this emoticon to your messages.

    After you have created an emoticon, you can add it to a message by typing its shortcut, or by selecting it from the emoticon menu in the message window.

Notes 

  • Depending on a contact's messaging application and messaging service, custom emoticons and some standard emoticons might not be visible to that contact.

  • The emoticon menu can display up to 59 emoticons.

  • To select which emoticons you want on the emoticon menu, on the Edit menu, click Emoticons, and then select or clear the check boxes.

Edit or delete a custom emoticon

If you created an emoticon, you can delete it or edit its name and keyboard shortcut.

  2. On the Edit menu, click Emoticons.
