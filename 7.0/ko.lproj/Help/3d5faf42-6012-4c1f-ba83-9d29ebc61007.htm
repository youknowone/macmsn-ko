I can't type characters for a certain language.

Show All

Cause: The correct keyboard input type is not turned on for the Office application that you are using.

Solution: 

Change the input type to the keyboard layout for the language that you want.

  1. On the Apple menu, click System Preferences.

  2. Under Personal, click International, and then click the Input Menu tab.

  3. In the list, select the Keyboard Viewer check box, and then select the check boxes for the keyboard input types that you want.

    Tip For more information about keyboard layouts, see Mac Help.

  4. Below the list, select the Show input menu in menu bar check box, and then close the International dialog box.

  5. Switch to the Office application that you want.

  6. On the right side of the menu bar, click the input menu, and then click the input type that you want.

    The input menu on the menu bar

    Tip To identify a character's location on the keyboard, click the input menu, and then click Show Keyboard Viewer.

Cause: The character is unavailable in the font that you are using.

Solution: Determine a font that contains the character, and then use that font to type the character.

  1. On the Apple menu, click System Preferences.

  2. Under Personal, click International, and then click the Input Menu tab.

  3. In the list, select the Character Palette check box.

  4. Below the list, select the Show input menu in menu bar check box, and then close the International dialog box.

  5. On the right side of the menu bar, click the input menu, and then click Show Character Palette.

    The input menu on the menu bar.

  6. On the View pop-up menu, click All Characters.

  7. Click the by Category tab, and then click the arrow next to the category that contains the language or script that you want.

  8. Click the language or script that you want, and then click the character that you want to check.

  9. Click the arrow next to Font Variation.

    Use one of the listed fonts to type the character that you want in the Office application that you are using.
