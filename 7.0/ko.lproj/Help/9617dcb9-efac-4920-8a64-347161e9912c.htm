Keyboard shortcuts for Messenger

You can use the following keyboard shortcuts to do tasks efficiently in Messenger.

To do this

Press

Sign in or sign out

⌘+G

Cancel sign in

⌘+.

Quit Messenger

⌘+Q

Show or hide the personal contact list

⌘+L

Show or hide the corporate contact list

OPTION+⌘+L

Close the active window (contact list or message window)

⌘+W

Display or hide the toolbar in the main or message window

⌘+T

Open the Conversation History

⌘+O

Open your e-mail application

⌘+E

Add a contact

SHIFT+⌘+A

Delete a selected contact

DELETE

View information about the selected contact

⌘+I

Save an instant message

⌘+S

Check spelling

SHIFT+⌘+:

Undo

⌘+Z

Cut

⌘+X

Copy

⌘+C

Paste

⌘+V

Select everything in the message window

⌘+A
