Mac OS X Universal Access features used in Office 2008

Microsoft Office applications use the universal access features of Mac OS X. Universal access features can make it easier to use your computer if it is difficult for you to see or hear or if you have limited dexterity.

To set custom preferences, use the Universal Access pane in System Preferences. Consult the following table for information about how Office applications work with the universal access features of Mac OS X.

Tab

Feature

Notes

Seeing

VoiceOver

Hear most menu commands, dialog boxes, and other elements on your computer screen. Does not work with the contents of the main document window. For example, VoiceOver does not read your text in a Word document.

Seeing 

Zoom

Zoom in on the screen image to make it appear larger.

Seeing

Display

Switch the screen to black on white or white on black, and make other contrast enhancements.

Hearing

Flash

Flash the screen when an alert sound occurs.

Keyboard

Sticky keys

Turn on sticky keys to help you press multiple keys and display pressed keys on the screen.

Office applications do not override the Mac OS X full keyboard navigation keyboard shortcuts that you can use to move to, select, or activate items on the screen.

Keyboard 

Slow keys

Turn on slow keys to help you get the computer to respond properly when you press keys.

Keyboard 

Key repeat rate

Change how fast a key repeats when pressed.

Mouse

Mouse keys

Turn on mouse keys to control the pointer by using the numeric keypad.

Mouse

Cursor size

Enlarge the pointer for increased visibility.

Mouse

Full keyboard access

Use the keyboard to move between controls instead of using the mouse.

For a list of full keyboard navigation keyboard shortcuts, see Mac Help.

Mouse

Tracking and scrolling speed

Change how fast the pointer moves across the screen and how fast scroll wheels function.
